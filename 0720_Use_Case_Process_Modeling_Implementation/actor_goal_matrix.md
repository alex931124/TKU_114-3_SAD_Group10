## 註記

- 系統管理員雖為章程中角色，但本期沒有新增正式權限模組，因此本次分析優先聚焦三個核心角色與系統保護行為。
- 本矩陣中的角色均位於系統邊界之外，避免把資料庫或畫面當作參與者。
